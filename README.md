# Code-Connect

Interface de uma rede social voltada para desenvolvedores que consome uma API para exibir postagens, interagir com publicações e compartilhar códigos e dicas de forma dinâmica.

#  🔨 Funcionalidades do projeto

O App permite visualizar o feed de publicações da comunidade dev, navegar de forma fluida entre diferentes páginas da plataforma, realizar login via autenticação segura com tokens e interagir com o conteúdo postado.
